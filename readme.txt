== WooCommerce integration in a few clicks ==

*Step 1*: Instructions for connecting a WooCommerce store to a SmokeDrop:
- In your WooCommerce account, generate a new API key
- Please note that the API Key must be generated by a user with Admin permissions
- How to generate a new API Key – https://docs.woocommerce.com/document/woocommerce-rest-api/#section-2
- SmokeDrop requires read/write permissions for the API key
- We recommend updating to the latest version of WooCommerce prior to generating your API key
- Log in to your Smoke Drop account
- From the homepage, go to Account
- Select General from the sidebar
- Select WooCommerce
- Enter your store name
- Enter your API key and API secret from WooCommerce
- Enter your store domain
- In most cases, you will need to include ‘https://www.’ and a forward slash at the end of your domain
- Select Save
- You’ve successfully connected your WooCommerce store to the app

== Installation ==

1. Upload the `SmokeDrop` folder to your `/wp-content/plugins/` directory or alternatively upload the SmokeDrop.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your computer.
2. Activate the SmokeDrop WordPress plugin through the 'Plugins' menu in WordPress.
3. Now that the plugin is activated, you will see SmokeDrop as an option in your WordPress sidebar. Select "SmokeDrop" to go to the connection page and click "Go To SmokeDrop Dashboard"
